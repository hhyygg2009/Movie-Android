# 看电影安卓版



#### 实现的功能
    1) 首页：负责展示热门内容

        1) 首页：把想展示电影直观的放出来给看的人
        2) NavigationView切换不同界面

    2) 列表页：负责列出、查找电影

        3) 水平电影列表，展示热门电影ScollView
        4) 电影的详细信息，可以标记想看和评分
        5) 给看过的电影评分
        6) 查找电影

    3) 个人中心，显示看过的电影

        7) 登陆账户
        8) 个人中心页面,可以设置个人信息
        9) 软件详情页
